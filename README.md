Tastenbelegung:
M = Toggle Trainer Menu
F6 = Toggle Clothing Menu
F5 = Toggle NoClip
H = Toggle Cam Mode in Vehicle
Q = UP
E = DOWN
W = MOVE FORWAR
S = MOVE BACK
D = MOVE RIGHT
A = MOVE LEFT
SCHIFT = Change Speed
